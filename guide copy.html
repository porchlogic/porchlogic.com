SMB1 Button Guide

Here's how to control your SMB1 with a single button. Use taps and holds to change modes, adjust latency, or check for updates.

Quick Actions

  • 1 tap: No action
  • 1 tap + hold: Decrease latency
  • 2 taps: No action
  • 2 taps + hold: Increase latency
  • 4 taps: Pairing mode

Accessing the Menu

Triple-tap the button (3 taps) to enter MENU mode. Then choose your option by tapping again:

  • 1 tap: Set mode to RX Clock
  • 2 taps: Set mode to RX Clock + Rhythm
  • 3 taps: Set mode to RX Clock + Melody
  • 4 taps: Set mode to TX (Transmitter)
  • 5 taps: Reboot & check for updates
