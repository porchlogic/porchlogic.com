# SMB1 Button Guide

<br>

>***example:***
>
>`🟢` = 1 quick press
>
>`🟢+hold` = press once and hold
>
>`🟢🔵🟡 .... 🟢` = 3 quick presses, *LED flashes*, then 1 quick press

---

<br>

| Number of presses | Result |
|-----------|--------|
| 🟢+hold    | ramp-down latency while held             |
| 🟢🔵+hold    | ramp-up latency while held          |
| 🟢🔵🟡🔴     | Enter Pairing Mode    |
| | |
| 🟢🔵🟡 .... 🟢     | switch to RX Mode            |
| 🟢🔵🟡 .... 🟢🔵     | *reserved*    |
| 🟢🔵🟡 .... 🟢🔵🟡     | *reserved*    |
| 🟢🔵🟡 .... 🟢🔵🟡🔴     | switch to TX Mode                  |
| 🟢🔵🟡 .... 🟢🔵🟡🔴🟠     | Check for Updates    |
