# SMB1 Button Guide




| Number of presses| Result |
|-----------|--------|
| 游릭+hold     | Decrease latency             |
| 游릭游댯+hold     | Increase latency             |
| 游릭游댯游리 .... 游릭     | RX Clock Mode            |
| 游릭游댯游리 .... 游릭游댯     | *reserved*    |
| 游릭游댯游리 .... 游릭游댯游리     | RX Clock + Melody    |
| 游릭游댯游리 .... 游릭游댯游리游댮     | TX Mode                  |
| 游릭游댯游리 .... 游릭游댯游리游댮游     | Check for Updates    |
| 游릭游댯游리游댮     | Enter Pairing Mode    |



---
***example:***

`游릭` = 1 press

`游릭游댯游리 .... 游릭` = 3 quick presses, *LED flashes*, then 1 quick press
