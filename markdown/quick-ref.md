# SMB1 Button Guide

<br>

>***example:***
>
>`游릭` = 1 quick press
>
>`游릭+hold` = press once and hold
>
>`游릭游댯游리 .... 游릭` = 3 quick presses, *LED flashes*, then 1 quick press

---

<br>

| Number of presses | Result |
|-----------|--------|
| 游릭+hold    | ramp-down latency while held             |
| 游릭游댯+hold    | ramp-up latency while held          |
| 游릭游댯游리游댮     | Enter Pairing Mode    |
| | |
| 游릭游댯游리 .... 游릭     | switch to RX Mode            |
| 游릭游댯游리 .... 游릭游댯     | *reserved*    |
| 游릭游댯游리 .... 游릭游댯游리     | *reserved*    |
| 游릭游댯游리 .... 游릭游댯游리游댮     | switch to TX Mode                  |
| 游릭游댯游리 .... 游릭游댯游리游댮游     | Check for Updates    |
