# SMB1 Button Guide




| Number of presses| Result |
|-----------|--------|
| 🟢+hold     | Decrease latency             |
| 🟢🔵+hold     | Increase latency             |
| 🟢🔵🟡 .... 🟢     | RX Clock Mode            |
| 🟢🔵🟡 .... 🟢🔵     | *reserved*    |
| 🟢🔵🟡 .... 🟢🔵🟡     | RX Clock + Melody    |
| 🟢🔵🟡 .... 🟢🔵🟡🔴     | TX Mode                  |
| 🟢🔵🟡 .... 🟢🔵🟡🔴🟠     | Check for Updates    |
| 🟢🔵🟡🔴     | Enter Pairing Mode    |



---
***example:***

`🟢` = 1 press

`🟢🔵🟡 .... 🟢` = 3 quick presses, *LED flashes*, then 1 quick press
